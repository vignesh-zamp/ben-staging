# **App Name**: Clutch Admin Panel

## Core Features:

- Authentication: Secure user authentication with email and password. The system will be pre-loaded with test credentials (benjamin@example.com / zamp123) for initial access and demo.
- Login Form: A clean, user-friendly login form with fields for email and password, and a 'Forgot password?' link.
- Admin Panel Access: Grant access to the admin panel upon successful login, routing to the dashboard (image 2).
- Task List Display: Display a list of tasks with details such as task name, status, due date, order, stock, customer, and assigned to.
- Sortable Columns: Enable sorting of tasks by clicking on column headers (Task, Status, Due Date, etc.).
- Clickable Rows: Make each task row clickable to view detailed information about the task.
- Filters and Columns: Provide options to filter and select which columns are visible in the task list.
- Search Functionality: Implement a search bar to search tasks by name, stock number, email, or task description.
- Tab Navigation: Implement tab navigation for 'My Tasks' and 'Saved Tabs'.

## Style Guidelines:

- Primary color: The main background will use a dark slate blue (#343A40), aiming to create a professional and focused ambiance.
- Background color: Slightly lighter slate blue (#495057) is used on the container at the top.
- Accent color: Sky blue (#A7D1FF) is used on the 'Login' button for high contrast against the darker background and drawing user attention.
- Body and headline font: 'Inter' for a clean, modern admin panel look and user interface.
- Centered login form on the page.
- Use of the existing logo 'clutch' as shown in the attached screenshots.
- Use a light gray or white for the background of the task list table for contrast and readability.
- Use blue for links and interactive elements in the task list.
- Use a clear and readable font for the task list table, such as 'Roboto' or 'Arial', with a font size of 14px.
- Ensure proper spacing and alignment of elements in the task list for a clean and organized look.
- Use icons to represent task status and actions, such as 'edit' and 'complete'.
